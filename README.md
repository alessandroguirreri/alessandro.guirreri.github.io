## **Cookie Management**

### *Technical Cookies (Always Active)*
- `consent_given`: User's choice (365 days)

### *Profiling Cookies (Consent Required)*
- `pref_sport`: Preferred sport (30 days)
- `fav_team`: Favorite team/athlete (30 days)
- `non_vis_location`: Approximate geolocation (90 days)
- `non_vis_age`: Random age for statistics (365 days)

Users can revoke consent anytime from the footer.
